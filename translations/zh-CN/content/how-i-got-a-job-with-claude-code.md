# 我如何用 Claude Code 得到一份全职工作

我受到[这篇帖子](https://www.reddit.com/r/ClaudeAI/comments/1pw14hj/claude_helped_me_get_a_full_time_job/)的启发，帖子提到他们使用 Claude Code 获得了一份全职工作，我也想在这里分享我的经历，希望能激励到一些人。

几个月前，旧金山的一家公司在 LinkedIn 上联系了我。我和他们进行了初步的电话筛选，之后又经历了一个月左右的面试流程。整个过程中，Claude Code 是最能帮助我的东西。

我首先用它做了初步的调研。我把公司官网的 URL 给了 Claude Code，让它浏览并总结发现。我还让它做一些网页搜索来做研究。我也让它看了一下他们的开源仓库，以便更好地理解他们的技术。

在最初的几次电话之后，他们让我做一个我并不熟悉的任务。这基本上是个数据工程的任务，而我从来不是数据工程师。它还应该是 AI 相关很重的，尽管我之前做过 AI，但我还没用过最适合这个任务的特定模型，特别是 CLIP。我有点犯难，而且只有几天时间准备。

但我想，好吧，把这个任务交给 Claude Code，看看我能用它做什么。我用语音转录系统与它配合来回沟通。一开始我的代码里有很多 bug，但最终我还是把一切都搞定了。最后，他们对我的项目印象深刻。我也有点不确定是否能在基本上一天内完成，但多亏了当时 Opus 4.1 的强大，我做到了。

此外，他们还让我介绍一篇我读过的博客文章。于是我想，好吧，干脆为此写一整篇博客吧？我照做了，他们也对此印象深刻。

总的来说，这就是我找到这份工作的过程：只是知道如何有效地使用 Claude Code。

从那以后，在过去的几个月里，我在工作中也大量使用它。它确实让我比原本更高效，也帮助我在当前的工作中脱颖而出。
