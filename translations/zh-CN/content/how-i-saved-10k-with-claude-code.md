# 我如何通过 Claude Code 省下 10,000 美元

几个月前开始看公寓准备购入时，我知道需要找到一个房产经纪。这对我来说是个相对陌生的区域，我在那儿也不认识好的经纪人。

所以我像其他人一样开始研究：用 Google 搜索。我找到了一些看起来不错的，给他们发了邮件。不过，没有一个足够好……

于是我决定用 Claude Code 做进一步研究。我让它帮我找出我想要的区域里最好的房产经纪，排除我已经联系过的。我要求它只提供能找到邮箱地址的那些。我让它把这些邮箱地址直接复制到我的剪贴板里，用逗号分隔。

整个过程我甚至没敲过字——100% 语音输入。

然后，我在密件抄送栏里发邮件给他们，说明我正准备买房。在总共 15 个“候选人”中，我根据他们的初次邮件回复进行了筛选，并和其中几位聊了聊。基于这些，我选定了一个我喜欢合作的。

时间快进几个月，我们找到了一个心仪的房源！当我告诉这位经纪人我喜欢这个房子时，他回答道：“我们要不要出价比挂牌价低 10%？”

我心里想，“谢天谢地，当然！” 我甚至不用提醒他先从低价开始——他就是知道该怎么做。

之后他不断给出最有竞争力的报价，并尽可能强硬地谈判。最终，他帮我们省下了 1 万美元。以他和卖方经纪谈判的激进程度，我不认为其他经纪人能帮我们省下这么多。

所以可以肯定地说，Claude Code 为我们节省了 1 万美元。订阅费是 100 美元——所以那个月给我带来了 100 倍的 ROI。
