# 我如何用 Claude Code 省下了 $10,000

几个月前我开始寻找要购买的公寓时，我知道自己需要找一位房产经纪人。对我来说那是一个相对陌生的区域，我不认识那里任何靠谱的经纪人。

所以我像其他人一样开始做调研：先用 Google 搜索。我找到了一些看起来不错的经纪人并给他们发了邮件。然而，他们最后都不够理想……

于是我决定用 Claude Code 做进一步调研。我让它帮我找出我关注区域里最好的房产经纪人，并排除我已经联系过的那些。我要求它只给出它能找到邮箱地址的人。我还让它把这些邮箱地址直接复制到我的剪贴板里，并用逗号分隔。

整个过程里我甚至一字未打——100% 语音输入。

然后，我把他们全部 BCC，并告诉他们我正在找房子买。在总共 15 位“候选人”中，我根据他们最初的邮件回复做了筛选，并和其中几位聊了聊。基于这些沟通，我选出了一个我喜欢合作的人。

现在快进到几个月后，我们找到了一个喜欢的房产！当我告诉这位经纪人我喜欢这个房产时，他的回应是：“我们要不要先出一个比挂牌价低 10% 的报价？”

我心里当时是：“谢天谢地，当然要！”我甚至都不用要求他先从低价开始——他自己就知道该怎么做。

之后他持续给出尽可能好的报价，并且尽可能强硬地谈判。结果是，最终他帮我们省下了 $10K。看他和卖方经纪人谈判时的强硬程度，我觉得换任何其他经纪人都不可能帮我们省下这么多。

所以可以肯定地说，Claude Code 帮我们省下了 $10K。它只花了我 $100 订阅费——所以那个月它给我的 ROI 是 100x。
